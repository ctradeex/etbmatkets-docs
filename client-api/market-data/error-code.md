# Error Code

### Error code description <a href="#cuo-wu-ma-shuo-ming" id="cuo-wu-ma-shuo-ming"></a>

| Error Code | meaning                       |
| ---------- | ----------------------------- |
| 200        | OK or partially OK            |
| 201        | request header param invalid  |
| 202        | request data param invalid    |
| 203        | client\_ip\_port is not exist |
| 500        | failed                        |
